**************
ast_decompiler
**************

ast_decompiler is a module for generating Python code given an AST.

A usage example::

    >> import ast
    >> from ast_decompiler import decompile

    >> decompile(ast.parse('(a + b) * c'))
    (a + b) * c

This module has been tested on Python 2.7, 3.3, 3.4, 3.5, 3.6, 3.7 and 3.8.
